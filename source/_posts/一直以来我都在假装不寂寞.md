# 再见，所有的福音战士

关于Eva，小的时候时间很多，本着尽善尽美的原则，我在网友的指导下先看了EVA的tv版和旧剧场版（死与新生、Air/真心为你）。但是因为反复收看的内容太多、时间太长，加上小时候的理解能力和情感感知能力有限，看完后虽然大感震撼，但也实在没有精力再去看新剧场版了。当时应该是高中，《新·福音战士剧场版：终》还没有发布。本着养到完结再看的宗旨，便一推再推，始终没能抽出时间来看新剧场版。

最近，《新·福音战士剧场版：终》重译为《天鹰战士：最后的冲击》被引入国内大荧幕，作为Eva系列的忠实~~假~~粉丝，我也是趁着这个周末补完了新剧场版《序》、《破》、《Q》，并补票前往电影院观看了《终》。

当影片进入最后一幕，为了彻底改写世界，真嗣举起盖乌斯之枪，贯穿自己与初号机。伴随那句台词——「さようなら、全てのエヴァンゲリオン」（再见了，所有的福音战士），所有的EVA相继被刺穿、消失。

{% image sayounara.jpg 再见了，所有的福音战士 %}

这不仅是真嗣的告别，也是庵野秀明对EVA、对观众、对过去的自己最后的道别。


# You Can (Not) Grow Up

旧版《EVA》里的真嗣，是被抛入成人世界的孩子。他没有选择，只能在痛苦中被迫成长。那种成长不是主动的，而是一种被现实碾压的结果——父亲的冷漠、同伴的疏离、世界的崩坏，所有的一切都在告诉他：“你必须长大。”然而，他拒绝。他沉默、逃避、害怕与他人接触。他既渴望被理解，又害怕被伤害。

而在《新剧场版》中，庵野秀明让真嗣重新获得了选择。新版的他依旧软弱，却不再是一个被动的受害者。他开始表达、尝试、甚至失败——但这一切都显得更真实，也更温柔。那些旧版中混乱、极端、带着厌恶与羞耻的片段被剔除（~~虽然有些给我造成强烈冲击的名场面被删除感觉还挺可惜的~~），取而代之的是更有尊严的成长。真嗣仍在逃避，但他学会了说出“我想要拯救她”、“我想再试一次”。这是一种更成熟的痛苦，是带着意识的成长。

{% image kiss.png 使我印象深刻的「大人的吻」 %}


这也是庵野自己的转变。旧版EVA像是一份创伤记录，是他对自身抑郁与孤独的公开解剖，也包含了对世界的恶意和报复；而新剧场版则更像康复日记，庵野事业有成、得到了老师和爱妻的支持，他不再自虐，而是试图与世界和解。庵野并没有否认痛苦，而是终于能用温柔去包裹它。

成长从来不是摆脱脆弱，而是允许自己在脆弱中继续生活。真嗣没有变得坚强，他只是终于能面对自己的软弱。You Can Grow Up——成长是痛苦的，但我们都可以学着像真嗣那样长大。

# 我们的碎片，补全成为一个完整的你

在EVA中，“补全”原本是一个宏大的计划：让所有人类的心灵融合，消除隔阂与痛苦，从而获得永恒的统一。但真嗣拒绝了这种没有界限的安宁——他宁愿承受痛苦，也要保留人与人之间的距离。于是，《终》中的补全变成了一种更私人、更温柔的实践：理解他人，从而理解自己。

剑介与冬二代表着一种“如果一切没发生”的可能性。他们没有被选中，也没有背负世界的重量。和他们在一起时，真嗣能短暂地成为一个“普通的少年”——开玩笑、一起吃饭、和他并肩走路、聊聊八卦。这些日常提醒他，他原来可以只是一个普通人，而不是eva驾驶员。

{% image Toji.jpg 冬二与班长结合，过上了普通但是幸福的生活 %}

绫波丽、明日香、渚薰是真嗣心理的三重投射。

绫波丽是母性与接纳的象征。绫波丽对真嗣无条件地接纳、无条件地爱，这是写入基因中的代码，正如母亲对自己的孩子一样。绫波丽在影片中不断消失、不断毁灭，实际上是真嗣在进行“精神断乳”。而最后当真嗣能微笑着面对不再需要他拯救的丽时，他也戒断了恋母情结，放下了对“被动被爱”的执念，从“想要被爱”走向“主动去爱”。

{% image Ayanami.jpg 与绫波丽告别 %}

明日香是自我厌恶与渴望认同的象征。她的傲慢、攻击性、竞争心，都是厚厚的防御。在与真嗣的关系里，表现为青春期最难处理的复合情绪：亲近的欲望+被拒绝的恐惧+自我羞耻。两个人都学会了同一种防御模式：假装不寂寞。明日香用冷漠和嘲讽掩藏自己的需要，真嗣用沉默与退缩压制自己的冲动。但是在《终》中，两人终于承认了自己的情感。明日香对真嗣说：“我曾经喜欢过你。”真嗣对明日香说：“谢谢你说曾经喜欢过我，我也喜欢过明日香哦。” 他们终于被一个具体的、真实的人认同，终于能够接纳自己的欲望。

{% image asuka.jpg 与明日香告别 %}

渚薰代表了无条件理解的幻象与灵魂的初恋。他对真嗣没有要求、没有评判，只是单纯地“喜欢他”。而正因为这份喜欢来自一个男性，它可以脱离传统意义上的浪漫关系，变得更像一种灵魂层面的接纳。他让真嗣第一次体验到一种超越性别、超越身体的“亲密感”。然而，渚薰对真嗣的“理解”没有边界、没有冲突、是一种理想化的理解幻象。所以他必须死，也只有他死后，真嗣才能理解：被理解不是永恒的状态，而是短暂的相遇。人不可能永远停留在“被无条件爱着”的世界；成长意味着接受“他人无法完全理解自己”的现实。

{% image nagisa.jpg 与渚薰告别 %}

葛城美里在旧版里被刻画得情绪化、矛盾、迷乱，但在新剧场版里，她不再逃避过去，而是学会带着恐惧去承担责任。她教会了真嗣如何在恐惧中仍然选择去爱、去承担、去保有继续活下去的勇气。加持为信念而死，美里在恐惧中学会承担——他们共同代表了“在不确定中仍选择前行”的成人勇气。

{% image Misato.jpg 葛城美里牺牲自己 %}

碇源堂是EVA世界中最沉默、最封闭的角色。他是“逃避型人格”的极端化版本。而真嗣作为其子延续了这一特点：两人都害怕被拒绝、害怕被伤害，因此选择了封闭自己。在电影最后，真嗣面对父亲时，没有选择逃避，反而向前走了一步，尝试理解父亲。他第一次意识到，父亲不是无所不能的神，只是一个痛苦的丈夫、一个懦弱的父亲、一个不知所措的男人。这正是“弑父”在心理意义上的完成：不是摧毁父亲，而是超越父亲。当碇真嗣能理解和怜悯碇源堂时，他已经不再被“父亲的影子”束缚，完成了与父亲的和解。而在与父亲和解的那一刻，他也终于完成了与自己的和解。

{% image ikari.jpg 与碇源堂告别 %}

最后是真希波。真希波并非旧版中延续而来的角色，而是新加入的角色，这正是她的意义所在：她是重置版宇宙的外来者，代表的是一种来自现实世界的视角。庵野自己在采访中多次提到，是妻子把他从抑郁的深渊中拉回现实。所以在电影的最后，也是真希波代表庵野妻子庵野梦洋子亲手为真嗣摘下了DSS项圈。这代表真嗣从“幻想的终结”走到了“现实的开始”——是时候结束幻想，走入现实的生活了。

{% image compare.png 庵野梦洋子和真希波 %}


# 当世界终于可以重新开始

在电影的最后，庵野让这一切归于现实：列车驶出站台，动画和实景的边界消失，虚构世界关闭，现实重新开始。

真嗣终于走出了循环。他不再逃避，也不再等待谁来拯救。他做出的选择是“让世界回到普通的状态”，这听起来很正常，但却是最艰难的。他明白了，不逃避并不意味着不痛苦，只是愿意带着痛苦去生活。“补全”最终的意义，不是让所有人融为一体，而是接受人与人之间无法消除的距离。

{% image new.jpg 长大后的真嗣携手真希波奔向新的生活 %}


庵野借这部电影完成了自己的和解。他从EVA走回现实，从创伤中走回生活。但是我却无法那样轻易地告别。看着真嗣和真希波离开站台，我知道真嗣得到了新的生活，而我还留在原地。我没有那样的结局。我的世界没有突然响起的One Last Kiss、没有绫波丽、没有明日香、没有渚薰，也没有真希波。

有的人能走出循环，有的人还在途中。不过，只要我们仍在尝试理解、尝试靠近，尝试不再逃避，不就已经很好了吗？
